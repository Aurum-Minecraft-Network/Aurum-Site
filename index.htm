Aurum Minecraft Network

Java Edition clients (Windows, MacOS, Linux)

To connect to the Aurum Minecraft Network server, log into Minecraft, select Multiplayer from the main menu, click the Add Server button, and enter the Java IP java.aurummcnet.ga:25690 as the IP address.

Note that to access the Aurum Minecraft Network server you need to run the same version of Minecraft as the latest version which GeyserMC supports (as of 24/02/2021, v1.16.5). If you need help with this, you can learn more in the minecraft.net article on Changing game version.

Bedrock Edition clients (Windows 10, Android, iOS, iPadOS)

To connect to the Aurum Minecraft Network Server, go to your Minecraft Bedrock client (e.g. Minecraft: Windows 10 Edition), press "Play", then go to the "Servers" tab. In the "Additional Servers" category, press "Add Server", then for the Server Address, enter the Bedrock IP bedrock.aurummcnet.ga, and for port, enter the Bedrock Port 25690. This may not be available for every Minecraft Bedrock client.

Bedrock Edition clients (Xbox One, Nintendo Switch)

To connect to the Aurum Minecraft Network Server, you will need an online service subscription for the console, which are Xbox Live Gold and Nintendo Switch Online respectively. Use BedrockConnect to connect to the server as follows:

Bedrock Edition Clients (PlayStation 4)

Unfortunately at this time, PS4 does not have any external server support. However, there are ways around this by emulating a LAN game on your network on another device.

Note that you will need a PlayStation Plus subscription.
Note that this method will also work with any other Bedrock platform other than Nintendo Switch.

Using a PC with a PlayStation 4

If you have a PC, you can use Phantom.

Using an Android Device with a PlayStation 4

If you have an Android device, you have several options:

  • Work in progress Geyser Android app
  • MC Lan Proxy (Trial)
  • MC Lan Proxy (Paid)
  • MC Server Connector

Using an iOS device with a PlayStation 4

If you have an iOS 14+ device, you can use BedrockTogether.

Enter the Bedrock IP bedrock.aurummcnet.ga for the Server Address, and enter the Bedrock Port 25690 for the Server Port.
